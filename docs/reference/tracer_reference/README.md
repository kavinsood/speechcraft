# Tracer Reference

This folder freezes the dirty tracer bullet that proved the dataset pipeline
math before productization.

Reference script:

```text
prove_the_math.py
```

It is intentionally not imported by the backend or dataset worker. The worker
implementation should preserve the tracer invariants while moving the logic
into smaller worker modules:

```text
analysis audio at 16 kHz
integer sample indices for cuts
processing buffers under 29.5 seconds
ASR/MFA on processing buffers
MFA-backed SafeCutPoints for training cuts
candidate review WAVs before export
```

Do not edit this file as product code. If behavior changes, update the worker
implementation and add/update worker tests.
